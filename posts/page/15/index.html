[DynamicsAX2009]AIFファイル上限値とその解除

#DynamicsAX2009 
デフォルト上限値の設定とエラー内容 DynamicsAXのAIFは外部ファイルとのやり取りを行うインターフェースの一つです。 外部ファイルとのやり取り時、明示されてはいませんがファイルサイズの上限値があります。 デフォルトでは10MBに設定されています。 これを超えるファイルのやり取りをAIFで行おうとすると下記エラーがイベントログに出力されます。 The adapter is unable to send the following message to Microsoft Dynamics AX: MessageId: XXXXXXXX-1111-AAAA-9999-ZZZZZZZZZZZZ Error: Error executing code: Insufficient memory to run script.
